# Fundamental Algorithm Techniques - Problem Set #9

Python verification scripts for finite functions, NAND logic equivalence, and boolean circuit complexity.

## Contents

1.  **Problem 1 (Finite Functions):** Verifies formulas for counting functions mapping $\{0,1\}^n$ to various output sets ($\{0,1\}$, $\{-1,0,1\}$, etc.).
2.  **Problem 2 (NAND Equivalence):** Implements `NOT`, `AND`, and `OR` gates using only `NAND` logic and verifies truth tables.
3.  **Problem 3 (Circuit Universality):** Explains and demonstrates why a function $F: \{0,1\}^n \to \{0,1\}$ requires a circuit size of $O(n \cdot 2^n)$.
